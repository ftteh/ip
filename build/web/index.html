Cash Web-App example

  • Responsive application
    • Twitter Bootstrap 3 CSS
    • Fixed top navigtion bar
    • Jumbotron for displaying important information
  • Custom bootstrap CSS - Gunmetal Slate from bootswatch
  • Custom icon - Glyphicons glyph included free in Twitter Bootstrap glyphicons package
    • How to use:
      • <span class="glyphicon glyphicon-home"></span>
  • MVC
    • (M)odel - JavaBeans
      • bean.Currency
      • bean.User
      • bean.CashRequest
    • (V)iew - JSP for:
      • input - HTML Form
      • output - data table, info and operation status
    • (C)ontroller - Servlet
  • JSP
    • get bean from session and use for page protection:
    • <c:if test="${sessionScope.adminprofile != null}">
    •     <% response.sendRedirect(request.getContextPath() + "/admin/terminate.html"); %>
    • </c:if>
    •  
    • get bean from page/request/session and display properties:
    • <jsp:useBean id="memberprofile" class="bean.User" scope="session" />
    • <jsp:getProperty name="memberprofile" property="fullName"/>
    •  
    • Set variable and increment value from ArrayList object:
    • <c:forEach items="${sessionScope.rqclist}" var="currentrqc" varStatus="loop">
    •     <c:set var="amountinprocess" scope="page" value="${0}"/>
    •     <c:if test="${currentrqc.status == 'in process'}">
    •         <c:set var="amountinprocess" scope="page" value="${amountinprocess + currentrqc.amount}"/>        
    •     </c:if>
    • </c:forEach>
    • <p>Amount in process: <c:out value="${amountinprocess}" /></p>
    •  
    • Create URL using beans properties and display the url in link:
    • <c:url value="crqapproval.jsp" var="displayURL">
    •     <c:param name="requestid" value="${currentrqc.requestId}" />
    • </c:url>
    • <a href="<c:out value='${displayURL}' />"><c:out value="${currentrqc.status}" /></a>
    •  
    • Create bean using GET method parameter:
    • <jsp:useBean id="crq" class="bean.RequestCash" scope="page">
    •     <jsp:setProperty name="crq" property="login" value="${param.login}"/>
    • </jsp:useBean>
